# ❤️ HeartGuard AI - Heart Disease Prediction System
![HeartGuard AI Screenshot](screenshot.png)

## 🚀 Overview

HeartGuard AI is a Machine Learning based web application that predicts the risk of heart disease using patient health parameters.

The project uses Logistic Regression and provides an interactive dashboard built with Streamlit.


## ✨ Features

- ❤️ Heart disease risk prediction
- 📊 Probability-based risk score
- 🎯 Machine Learning prediction
- 📈 Interactive risk gauge
- 🎨 Modern Streamlit dashboard
- ⚡ Real-time prediction


## 🧠 Machine Learning

Algorithm Used:

- Logistic Regression


### Input Features:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise Angina
- Oldpeak
- ST Slope
